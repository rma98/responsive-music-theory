Explicações sobre os Graus

Acordes Maiores

  • 1º Grau - Tônica: Ponto de resolução e descanso.
  • 4º Grau - Subdominante: Prepara a resolução.
  • 5º Grau - Dominante: Gera tensão para resolver na Tônica.

Acordes Menores

  • 2º Grau - Subdominante menor: Transição suave para a Dominante.
  • 3º Grau - Mediante: Um acorde menor, com função intermediária.
  • 6º Grau - Tônica menor relativa: Pode ser usado para mudar o modo para menor.

Acorde Diminuto

  • 7º Grau - Subtônica diminuta: Tensão extrema, pedindo resolução imediata.

Perguntas Frequentes (FAQ)

O que é um Campo Harmônico?

O Campo Harmônico é um conjunto de acordes formados a partir de uma escala musical. Ele segue uma ordem específica de intervalos, proporcionando uma base harmônica coerente.

Como posso usar o Campo Harmônico para compor músicas?

Utilizando os acordes do Campo Harmônico, você pode criar progressões harmônicas que fazem sentido dentro de uma tonalidade, gerando músicas harmoniosas e coesas.

Qual a diferença entre acordes maiores e menores?

Acordes maiores são geralmente associados a sonoridades mais alegres e brilhantes, enquanto acordes menores tendem a evocar emoções mais melancólicas ou introspectivas.
